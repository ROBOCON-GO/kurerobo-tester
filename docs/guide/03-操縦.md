# ③ 操縦と制御プログラム(control)

DualSense でロボットを操縦します。ロボットの動きは `control(pad, motor)` という関数に書きます。
**編集できるのはこの関数だけ**で、通信・安全停止などの仕組みは触れないようロックされています。

## 使い方

1. ②と同様に「接続」(Bluetooth仮想COM)
2. **DualSense をPCに接続**(USB または Bluetooth)し、何かボタンを押してから「**コントローラ接続**」
   - 「LX=… LY=…」のライブ表示が動けばOK
3. エディタの `control()` を必要に応じて編集
4. 「**適用(コンパイル)**」を押す → `✓ 適用しました` が出れば反映
5. スティック/ボタンで操縦

> 編集中にエラーがあると `✗ エラー: …` と表示され、その間は安全のため送信を止めます。
> 実行中にエラーが起きた場合も自動で送信停止します。

## プロファイル(保存・読み込み)

- **保存**: 名前を付けて control() を保存(ロボットごとに切り替え可能)
- **書出(.js)**: ファイルとして書き出し(共有・バックアップ)
- **読込**: ファイルから読み込み

## 関数リファレンス(お品書き)

### 入力 `pad`
| 書き方 | 内容 |
| --- | --- |
| `pad.LX` / `pad.LY` / `pad.RX` / `pad.RY` | 左右スティックのX/Y。範囲 **-3000〜3000**。中央付近(±500)は0(デッドゾーン) |
| `pad.pressed("△")` | ボタンが押されていれば true |
| `pad.buttons` | 押下状態のビット列(整数) |

ボタン名: `× ○ □ △ L1 R1 L2 R2 share options L3 R3 ↑ ↓ ← →`

### 出力 `motor`
| 書き方 | 内容 |
| --- | --- |
| `motor.speed(id, rpm)` | ロボマス速度(M2006/M3508, id 1〜8) |
| `motor.phase(id, 目標, 速度?)` | M2006/M3508 **速度指定付き位相制御**。目標=回転数。`-1`で無効(その場停止) |
| `motor.gm6020(id, val)` | GM6020 速度(id 1〜7) |
| `motor.gm6020Phase(id, deg, 速度?)` | GM6020 **速度指定付き位相制御**。deg=角度(360で1回転)。`-1`で停止 |
| `motor.brushed(id, duty)` | 新モタドラ(id 1〜8, -999〜999, 0=ブレーキ) |
| `motor.pwm(ch, pulse)` | PWM(ch 1〜6)。絶縁反転は自動 |
| `motor.gpio(state)` | GPIO 出力ビット列 |

- 位相制御の**速度を省く**とメインボード既定(8000)で動きます。
- 位相は**原点(リセット時点)基準**です。

## 例

```js
function control(pad, motor) {
  // 4輪オムニ:右スティックで全方向、左スティックXで旋回
  const o1 = pad.RX - pad.RY + pad.LX;
  const o2 = pad.RX + pad.RY + pad.LX;
  const o3 = -pad.RX + pad.RY + pad.LX;
  const o4 = -pad.RX - pad.RY + pad.LX;
  motor.speed(1, o1);
  motor.speed(2, o2);
  motor.speed(3, o3);
  motor.speed(4, o4);

  // △を押している間 新モタドラID1を回す(離すとブレーキ)
  if (pad.pressed("△")) motor.brushed(1, 500);
  else                  motor.brushed(1, 0);

  // ○を押している間 ID5を「5回転」位置へ速度4000で(離すと停止)
  if (pad.pressed("○")) motor.phase(5, 5, 4000);
  else                  motor.phase(5, -1);

  // ×を押している間 GM6020 ID1を90度へ速度500で(離すと停止)
  if (pad.pressed("×")) motor.gm6020Phase(1, 90, 500);
  else                  motor.gm6020Phase(1, -1);
}
```

## コツ・注意

- **押している間だけ動かす**なら `if (押されている) {動かす} else {止める}` の形にする。
  else を書かないと、一度動かすと止まりません。
- スティックのような常時入力は `if` に入れず、そのまま毎回呼びます。
- 位相制御で「行き過ぎ→急反転」は機構を壊すことがあるので、目標値の範囲に注意。
