# KureRobo Tester ドキュメント

呉ロボ用、ブラウザ完結型のロボットテストツールの使い方。
**ノートPC + DualSense + ESP32 + ロボマス開発ボード(mawarudokusute)** でロボットを動かせます。

## これは何?

「誰でも簡単にロボットを動かす」ためのツールです。Arduino IDE も C言語のコンパイルも不要で、
**ブラウザ(Chrome/Edge)を開くだけ**でモーターを回したり、コントローラで操縦したりできます。

3つのタブがあります:

| タブ | 何ができる | 主な利用者 |
| --- | --- | --- |
| ① 書き込み | ESP32 にファームを書き込む(最初に一度だけ) | 制御班 |
| ② 手動テスト | スライダーでモーター単体テスト(コード不要) | 機械班 |
| ③ 操縦 | DualSenseで操縦。動きは `control()` に書く | 制御班 / 慣れた人 |

## 仕組み(ざっくり)

```
[ブラウザ] --Bluetooth--> [ESP32(素通しブリッジ)] --UART--> [mawarudokusute] --CAN/PWM--> モーター
```

- 制御の計算はすべて**ブラウザ側**で行います。ESP32 は受け取ったデータをそのまま流すだけ。
- なので ESP32 のファームは一度書けば**二度と書き換え不要**です。
