ESP32 ブリッジファーム書き込み

インストール不要のポータブル書き込みツール(esptool同梱)。ダウンロードして解凍し、flash.bat をダブルクリックするだけです。

⬇ 書き込みツール(ZIP・約14MB)をダウンロード

  1. ZIPをダウンロードして解凍(右クリック→すべて展開)
  2. USBで ESP32 を接続する
  3. 解凍フォルダ内の flash.bat をダブルクリック
  4. 黒い画面に OK! Flash complete (hash verified) が出れば完了(COMは自動検出)

※ Windowsが「PCを保護しました」と出たら「詳細情報 → 実行」(同梱 esptool.exe が署名なしのため)。
※ ブリッジは固定ファームなので、各ESP32は一度書けば以後の書き込みは不要です。
※ ブラウザからの直接書き込み(esptool-js)は、このボード(CP2102)でフラッシュを破損させたため不採用です。

接続中は 50ms 周期で送信し続けます(メインボードの800ms安全リセット対策)。行を有効化してスライダーを動かしてください。

関数のお品書き(クリックで開閉)

入力 pad

pad.LX / LY / RX / RY左右スティック (-3000〜3000, デッドゾーン適用済)
pad.pressed("△")ボタン判定(true/false)
pad.buttons押下ビット列(整数)

ボタン名: × ○ □ △ L1 R1 L2 R2 share options L3 R3 ↑ ↓ ← →

出力 motor

motor.speed(id, rpm)ロボマス速度 (M2006/M3508, id 1〜8)
motor.phase(id, 目標, 速度?)M2006/M3508 速度指定付き位相。目標=回転数 / -1で停止
motor.gm6020(id, val)GM6020 速度 (id 1〜7)
motor.gm6020Phase(id, deg, 速度?)GM6020 速度指定付き位相。deg=角度(360で1回転) / -1で停止
motor.brushed(id, duty)新モタドラ (id 1〜8, -999〜999, 0=ブレーキ)
motor.pwm(ch, pulse)PWM (ch 1〜6, 絶縁反転は自動)
motor.gpio(state)GPIO 出力ビット列

速度を省くと既定(8000)で動作。位相は原点(リセット時点)基準。
control()の外に書いた変数は「適用」時に1回だけ初期化され、以後フレーム間で保持されます(状態保持OK)。
